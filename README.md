## Usage

### Requirements
- Rust -- [installation](https://www.rust-lang.org/tools/install)
- Trunk -- [installation](https://trunkrs.dev/#install)

### Backend
Run the server.
```bash
cargo run --bin backend
```

### Frontend
Serve the frontend.
```bash
cd frontend && trunk serve
```

Finally, visit `localhost:8080` in your browser
